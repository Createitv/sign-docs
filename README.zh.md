# SignDocs — 文档签章编辑器

[![Release](https://img.shields.io/github/v/release/Createitv/sign-docs)](https://github.com/Createitv/sign-docs/releases/latest)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

[English](README.md)

SignDocs 是一款桌面应用，用于打开 PDF 和 Word 文档，在上面添加电子公章和手写签名，并将结果导出为 PDF 文件。所有操作均在本地完成，无需上传、无需联网。

## 能做什么

- 打开 PDF 和 Word（DOCX/DOC）文档
- 在文档任意位置加盖公章——可上传自己的图片，也可在应用内生成圆形电子公章
- 手写签名并拖放到需要的位置
- 添加可移动的文本框，用于填写或标注信息
- 将完成后的文档导出为 PDF 文件
